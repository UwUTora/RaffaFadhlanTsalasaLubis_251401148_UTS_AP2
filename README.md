## 🎥 Ketentuan Video (Penting)

Selain file coding, wajib melampirkan **video penjelasan**.

Isi video minimal:

### Untuk soal utama:
- Menjalankan program
- Menunjukkan output
- Menjelaskan singkat alur program
- Memperlihatkan wajah dari awal sampai akhir

### Untuk soal unik:
Kerjakan sesuai perintah

✅ Jika diminta mengetik ulang, harus tampilkan video kamu mengetik ulang sambil di rekam

✅ Jika diminta menjelaskan, maka jelaskan dalam video
